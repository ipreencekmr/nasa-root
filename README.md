## nasa-root

> [One App](https://github.com/americanexpress/one-app#--) `holocron` module.

## Prerequisite

- Node
- Docker

## Usage 

```bash
git clone https://github.com/ipreencekmr/nasa-root.git
cd nasa-root
npm install
npm run build
npm run start
```

## Available Scripts

`npm run test`

`npm run build:watch`

`npm run start`

`npm run start:test`

`npm run generate:map`
